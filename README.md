# 《马路指挥家》

这是一款看起来像普通过马路、玩起来却会不断失控的轻策略街机游戏。

你和一名 AI 引导员分别站在街道两侧，目标是指挥小朋友安全穿过车流。每成功送达一名小朋友，就会触发一条临时新规则，让整场比赛立刻发生变化。

规则可能会让车辆变慢、指挥范围扩大、小朋友一次成群过街，也可能让局势突然偏向另一方。更特别的是：玩家可以在送达后亲自输入规则，让 AI 根据你的想法生成可执行的新规则，并立刻应用到接下来的游戏中。

## 核心玩法

- 指挥小朋友前进或停止，避开不断驶来的车辆。
- 和 AI 对手竞争，看谁能更快、更安全地送达更多小朋友。
- 每次送达都会改变规则，让比赛持续产生新的节奏。
- 玩家可以输入自定义规则，直接改写当前街道局势。
- 场上最多同时存在多条临时规则，局势会越来越混乱，也越来越有趣。

## 游戏特色

- 熟悉的过马路玩法，不熟悉的规则变化。
- AI 对手会主动寻找、护送、判断车流并参与竞争。
- 玩家制定的新规则会影响接下来的比赛。
- 临时规则持续时间有限，适合制造反转和机会。
- 每一局都会因为规则组合不同而产生新的体验。
